How to compile the documentation
--------------------------------

0. Install sphynx: 
> `pip install sphinx`
1. Navigate to 01_Docs
> `cd 01_docs`
2. Compile the doc
> `make html`

Docstring convention
--------------------

OFF documentation follows the [Numpy doctring style guide](https://numpydoc.readthedocs.io/en/latest/format.html)
